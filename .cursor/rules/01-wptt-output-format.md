# Output format (mandatory)

When asked to implement or document anything, output must include:

## Context
- module/entrypoint paths

## Impact layer
- choose the affected layer(s)

## State changes
- exact files and state keys/markers changed

## Rollback
- how to revert and which services to restart

## Verify
- minimal commands/checks to confirm success

If information is missing, state **minimal assumptions** and proceed safely.
